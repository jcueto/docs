    # oasis.setGateway

    The oasis.setGateway function configures the client so that all service communication is done through the given OasisGateway. This method should be called before interacting with any services.

    oasis.setGateway(gateway)

    # Parameters

    1. gateway - OasisGateway: The gateway to facilitate all service communications.
